# JS Internals Visualizer (Next.js App Router)

Interactive learning app to visualize how JavaScript works internally.

## Features
- Event loop, call stack, Web APIs, callback queue simulation
- Closures and lexical scope memory
- Prototype chain lookup walk-through
- `this` binding behavior playground
- UI animations with Framer Motion and GSAP

## Run locally
```bash
npm install
npm run dev
```

Open http://localhost:3000
